# Práctica 01 - HackerRank

## Objetivo
Practicar conceptos básicos de Java como entrada/salida, herencia y sobrecarga de métodos.

## Ejercicios incluidos
1. **Welcome to Java!**  
   - Escritura en salida estándar.  
   - Ejecución:
     ```bash
     javac Welcome_to_Java/Solution.java
     java Welcome_to_Java/Solution.java
     ```

2. **Java Stdin and Stdout I**  
   - Lectura de datos desde consola y escritura en salida estándar.  
   - Ejecución:
     ```bash
     javac Java_Stdin_and_Stdout_I/Solution.java
     java Java_Stdin_and_Stdout_I/Solution.java
     ```
